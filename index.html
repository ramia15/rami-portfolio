Easy QR Code Generator

Understanding the increasing need for QR codes in the digital era, I developed the Easy QR Code Generator. This tool not only generates QR codes but also has an integrated URL shortener for added convenience. Recognizing the importance of aesthetics, I implemented features that allow users to customize the QR code's size and colors. Additionally, the Dark Mode ensures a comfortable user experience, while the contrast warning ensures the generated QR codes are always readable. Deployed using Heroku.
